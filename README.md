# IPL Statistics & Performance Analytics System

## 📊 Overview
This project is an interactive Power BI dashboard that analyzes IPL data across seasons, teams, and players.

## 🚀 Features
- Season-wise performance analysis
- Player & Team comparison
- Average Runs per Season with tooltip insights
- Wickets leaderboard & top scorers
- Interactive filters, slicers, and drill-through

## 🛠 Tools Used
- Power BI
- Data Modeling
- DAX
- Data Cleaning



## 🎯 Use Case
Helps analyze player consistency, team performance, and match trends.
